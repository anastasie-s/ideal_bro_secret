# IDEAL BRO site

Статичный сайт для GitHub Pages.

Страницы:
- index.html — главная
- test.html — кастомный тест
- rules.html — правила
- peak.html — скрытая страница с кодом доступа, не добавлена в меню

Коды и задания меняются в peak.js.

Важно: пароль на GitHub Pages — не настоящая серверная защита. Для ивента и защиты от случайного просмотра подходит, но задания технически можно увидеть в коде страницы.
